Projects

Virtual CPU Simulator & Disassembler

Solo Developer

I developed a virtual CPU simulator using C for simulating register behavior, memory manipulation, and control flow to dynamically interpret machine code and update the CPU's state. Additionally, I implemented a Y86-64 disassembler utilizing function pointers, structs, and unions to handle instruction decoding efficiently.

The project also involved performance optimizations, such as cache-aware algorithms and memory optimizations, to leverage spatial and temporal locality. This experience deepened my understanding of low-level hardware interaction and optimization strategies.

Specialized Calculator

Developer/Scrum Master

Huffman File Compressor

Solo Developer

I designed a Huffman encoder and decoder in Java. This project implemented core classes such as BitSequence for managing sequences of bits, HuffmanNode to represent tree nodes with frequency counts, and HuffmanTree for the main encoding and decoding functions. Additionally, the project included utility classes for file handling and a graphical user interface (GUI) for compressing and decompressing files using the Huffman algorithm.

The result was a fully functional compression tool that allowed users to zip and unzip files efficiently using Huffman encoding.
